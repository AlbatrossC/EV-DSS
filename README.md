# EV Transition & Analysis Platform

This project is a comprehensive decision support system for analyzing the transition from ICE to EVs.

## Getting Started

### Prerequisites
- Node.js & npm

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Technologies
- vite
- React
- Typescript
- Tailwind CSS
- shadcn/ui
- Supabase
